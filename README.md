# Wiley & Sons Project Management Plan
# Overview

This repository contains project planning and scheduling deliverables developed for the Wiley & Sons project using Microsoft Project. The project demonstrates core project management practices including scheduling, task sequencing, dependency mapping, resource allocation, milestone tracking, and critical path analysis.

The purpose of this project was to develop an integrated project schedule and apply project management methodologies to support effective planning, coordination, and project execution.

# Project Objectives
Develop a structured project schedule
Define task dependencies and sequencing
Allocate and manage project resources
Identify project milestones and deliverables
Analyze the project critical path
Apply project scheduling best practices using Microsoft Project
